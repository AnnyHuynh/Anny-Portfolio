Word-Guess-Game


I used CSS, HTML to create the simple template and design on the game page. I used JavaScript to make the page respond to the user’s guesses. For JavaScript, I used For Loop to loop all the animal’s names and put them into an array. Then I used function, “if...else if” statements to respond to user’s guesses.

In this game, user has 10 guesses to guess the letters of an animal’s name. if the user guesses the letter correctly, the letter will get display into the name box. If the letter is incorrect, it will be listed under the “letters that already guessed” portion. Once the animal’s name is completely filled with all the correct letters, the animal image will be displayed and you win the game.

Crystals Game


I used HTML, CSS to create template and design on the game page. I also used jQuery (jQuery events, selectors, etc..), JavaScript Random() Method to generate the computer guesses and respond to user guesses.

In this game, user will be given a random number at the start of the game. There are four crystals below. By clicking on a crystal user will add a specific amount of points to the total score. User win the game by matching user’s total score to random number, user lose the game if user’s total score goes above the random number.

Trivia Game


I used HTML, CSS to create the template and design on the page and used JavaScript for the logic. For JavaScript, I used JS Objects to set variables, if statements, and JS functions (timing function, stop function, reset function, etc…) to make the page responsive.

In this quiz, user will have 60 seconds to complete the quiz. When user click "Start" button, the timer will start count down. User will have total 10 questions. Click "Submit" button at the bottom of the page after done. If the timer is at 0 second, user’s time is out and answers will be submitted automatically. The result page will show user results after click "Submit" or after the time is out. Click "Reset" button to restart the game.

Travel-the-World


I used HTML, CSS to create the template and design on the page. I used jQuery AJAX method (used GIPHY API in query URL in AJAX to make a dynamic web page that populates with gifs.) On click event function is used to animate the still GIPHY images. If the user clicks the gif again, animation will stop playing.

After you click on a country button, images about that country will be generated below the buttons. Click on an image, you will see the animation of that image. Click on that image again to pause the animation. You can add a different country that are not on the list by entering a country name into The input box, then click add country.

Train Scheduler


I used Bootstrap, HTML, CSS to create the template and design on the app. I used Firebase to host arrival and departure data of the train schedules. I used Moment.js to retrieve or manipulate the time, and also to calculate when the next train will arrive which is relative to the current time.

Check out your time Schedule by filling out the form. Form include: Train Name, Destination you want to go, first train start, and how frequency. After filling the form, click submit and you should be able to see your next time Train and how many minutes away until the next train come. Have a good trip!

Liri-Bot-App


Liri-Node-App is a Song, Movie, and Concert search engine. You can search songs, movies, and concerts using this application with Node.js. This application is using Spotify API, OMDB API, and BandsInTown API to search the related information. This application also use moment.js to format the dates on searches.

Go to Bash/Terminal and type in:
node liri.js spotify-this-song "Your-Song-Name"
node liri.js movie-this "Your-Movie-Name"
node liri.js concer-this "Your-Artist's-Name"
